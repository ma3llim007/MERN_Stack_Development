Types Of Events

  1. onchange - An HTML element has been changed
  2. onclick - The user clicks an HTML element
  3. onmouseover - The user moves the mouse over an HTML element
  4. onmouseout - The user moves the mouse away from an HTML element
  5. onload - The browser has finished loading the page
  6. abort - The loading of a media is aborted
  7. afterprint - A page has started printing
  8. animationend - A CSS animation has completed
  9. animationiteration - A CSS animation is repeated
  10. animationstart - A CSS animation has started
  11. beforeprint - A page is about to be printed
  12. beforeunload - Before a document is about to be unloaded
  13. blur - An element loses focus
  14. canplay -The browser can start playing a media (has buffered enough to begin)
  15. canplaythrough - The browser can play through a media without stopping for buffering
  16. change -The content of a form element has changed
  17. click - An element is clicked on
  18. contextmenu - An element is right-clicked to open a context menu
  19. copy - The content of an element is copied
  20. cut The content of an element is cut
  21. dblclick An element is double-clicked
  22. drag An element is being dragged
  23. dragend Dragging of an element has ended
  24. dragenter A dragged element enters the drop target
  25. dragleave A dragged element leaves the drop target
  26. dragover A dragged element is over the drop target
  27. dragstart Dragging of an element has started
  28. drop A dragged element is dropped on the target
  29. durationchange The duration of a media is changed
  30. ended A media has reach the end ("thanks for listening")
  31. error An error has occurred while loading a file.
  32. focus An element gets focus
  33. focusin An element is about to get focus
  34. focusout An element is about to lose focus
  35. fullscreenchange An element is displayed in fullscreen mode
  36. fullscreenerror An element can not be displayed in fullscreen mode
  37. hashchange There has been changes to the anchor part of a URL HashChangeEvent
  38. input An element gets user input InputEvent
  39. invalid An element is invalid Event
  40. keydown A key is down KeyboardEvent
  41. keypress A key is pressed KeyboardEvent
  42. keyup A key is released KeyboardEvent
  43. load An object has loaded UiEvent, Event
  44. loadeddata Media data is loaded Event
  45. loadedmetadata Meta data (like dimensions and duration) are loaded
  46. loadstart The browser starts looking for the specified media
  47. message A message is received through the event source
  48. mousedown The mouse button is pressed over an element
  49. mouseenter The pointer is moved onto an element
  50. mouseleave The pointer is moved out of an element
  51. mousemove The pointer is moved over an element
  52. mouseover The pointer is moved onto an element
  53. mouseout The pointer is moved out of an element
  54. mouseup A user releases a mouse button over an element
  55. mousewheel Deprecated. Use the wheel event instead
  56. offline The browser starts working offline
  57. online The browser starts working online
  58. open A connection with the event source is opened
  59. pagehide User navigates away from a webpage
  60. pageshow User navigates to a webpage
  61. paste Some content is pasted in an element
  62. pause A media is paused Event
  63. play The media has started or is no longer paused
  64. playing The media is playing after being paused or buffered
  65. popstate The window's history changes
  66. progress The browser is downloading media data
  67. ratechange The playing speed of a media is changed
  68. resize The document view is resized
  69. reset A form is reset
  70. scroll An scrollbar is being scrolled
  71. search Something is written in a search field
  72. seeked Skipping to a media position is finished
  73. seeking Skipping to a media position is started
  74. select User selects some text
  75. show
  76. stalled The browser is trying to get unavailable media data
  77. storage A Web Storage area is updated
  78. submit A form is submitted
  79. suspend The browser is intentionally not getting media data
  80. timeupdate The playing position has changed (the user moves to a different point in the media)
  81. toggle The user opens or closes the
    element
  82. touchcancel The touch is interrupted
  83. touchend A finger is removed from a touch screen
  84. touchmove A finger is dragged across the screen
  85. touchstart A finger is placed on a touch screen
  86. transitionend A CSS transition has completed
  87. unload A page has unloaded
  88. volumechange The volume of a media is changed (includes muting)
  89. waiting A media is paused but is expected to resume (e.g. buffering)
  90. wheel The mouse wheel rolls up or down over an element
